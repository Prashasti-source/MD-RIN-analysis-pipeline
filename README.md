# FUS-RIN-analysis

Python pipeline for residue interaction network (RIN) analysis of molecular dynamics trajectories from ALS-associated FUS variants.

## Features

- Cα-based residue interaction network generation
- Betweenness centrality analysis
- Comparative network topology analysis
- Heatmap visualization
- Smoothed communication profiles
- Automated processing of MD trajectories

## Requirements

- Python 3.x
- MDAnalysis
- NetworkX
- NumPy
- Matplotlib
- Seaborn

## Installation

```bash
pip install MDAnalysis networkx numpy matplotlib seaborn
